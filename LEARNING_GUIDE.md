# Learning Guide

Use `ai-infra-senior-architect-solutions` after attempting the paired learning material in `ai-infra-senior-architect-learning`.

## Recommended Workflow

1. Attempt the exercise or project in `ai-infra-senior-architect-learning` first.
2. Compare your approach with the matching reference solution here.
3. Note the differences in structure, testing, error handling, and operations.
4. Rebuild or extend your own implementation after reviewing the reference.

## Repository Layout

- `modules/`: module-by-module reference solutions
- `projects/`: project and capstone implementations
- `guides/`: cross-cutting debugging, optimization, and production notes
- `resources/`: shared setup notes and supporting material

## Ground Rules

- Do not use this repo as a copy-paste answer bank.
- Prefer understanding why a solution is structured a certain way before reusing it.
- Keep module slugs aligned with the paired learning repository.
