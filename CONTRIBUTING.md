# Contributing to `ai-infra-senior-architect-solutions`

This repository stores reference implementations for `ai-infra-senior-architect-learning`.

## What belongs in this repository

- module-level reference solutions in `modules/`
- project-grade reference implementations in `projects/`
- cross-cutting explanations and troubleshooting notes in `guides/`
- shared references and support material in `resources/`

## Repository rules

- Keep module and project slugs aligned with `ai-infra-senior-architect-learning`.
- Maintain 1:1 solution coverage for learner exercises and projects.
- Prefer runnable, documented solutions over status summaries.
- Keep operational reports and housekeeping notes out of the repo root.

## Pull request checklist

- Preserved path parity with the learning repository.
- Added or updated documentation for new solutions.
- Checked links and runnable entry points.
- Avoided root-level housekeeping clutter.
